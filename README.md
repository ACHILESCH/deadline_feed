# DEADLINE_FEED

**DEADLINE_FEED** is a structured academic task management system designed to model real academic workflow — subjects, chapters, and deadline-driven tasks — in a clean, distraction-free interface.

---

## Purpose

Most study planners are simple to-do lists.

**DEADLINE_FEED** introduces:

- Academic hierarchy (Subject → Chapter → Task)  
- Visual urgency logic  
- Smart deadline indicators  
- Structured task creation flow  
- Clean, framework-free architecture  

---

## Features->

- Profile setup & edit  
- Subject management  
- Chapter management per subject  
- Dynamic Subject → Chapter dropdown filtering  
- Interactive urgency rating system  
- Smart deadline color indicators  
- Auto-move completed tasks  
- Right-click context menu (Edit/Delete)  
- Persistent Dark / Light mode  
- Smooth UI transitions  
- LocalStorage data persistence  
- Fully offline capable  

---

## Built With

- HTML5  
- Modern CSS (variables + animations)  
- Vanilla JavaScript  
- LocalStorage API  

_No frameworks. No AI. No dependencies._
